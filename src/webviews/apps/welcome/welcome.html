GitLens

Git 增强版

GitLens 可在 VS Code 中显著增强 Git 体验，帮助你挖掘每个仓库中的历史信息。它能通过 Git 归属注释和 CodeLens 让你快速看清代码作者信息，顺畅浏览和探索 Git 仓库，并借助丰富的可视化与强大的比较命令获得更多洞察。

欢迎使用 GitLens 12

查看 GitLens 12 新功能 或阅读发行说明 打开入门引导 或了解 GitLens+ 功能

快速设置

GitLens 功能强大、特性丰富，并且高度可定制。若你觉得 CodeLens 过于显眼，或当前行归属注释 注解会分散注意力，也没关系，可以直接通过下面的选项关闭它们，或快速调整它们的行为方式。

如需更多选项，请使用交互式 GitLens 设置编辑器

在当前行末尾添加低干扰的归属注释

在文件顶部和代码块上显示作者信息 CodeLens

在状态栏中显示当前行的 Git 归属信息
